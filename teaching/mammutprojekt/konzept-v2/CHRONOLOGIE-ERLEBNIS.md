# Chronologie der Erlebnisse 1871-1918
## Struktur für immersives Spieldesign

**Prinzip:** Jedes historische Ereignis wird auf Erlebnis-Ebene heruntergebrochen.
- **Makro** = Was Historiker schreiben (Vogelperspektive)
- **Mikro** = Wie Menschen es erlebten (Froschperspektive)
- **Quelle** = Woher wir das wissen

## 1878 — Beispieljahr (detailliert)

### Politik

#### Sozialistengesetz (21. Oktober 1878)
**Makro:** Bismarck verbietet SPD, Gewerkschaften, Arbeiterpresse nach zwei Attentaten auf Kaiser Wilhelm I.

**Mikro — Arbeiter-Perspektive:**
- [ ] Die "Vorwärts"-Zeitung erscheint nicht mehr. Der Wirt im Stammlokal versteckt die letzten Ausgaben unterm Tresen.
- [ ] Polizei durchsucht Wohnungen nach "sozialistischem Schriftgut"
- [ ] Versammlungsverbot: "Mehr als 5 Arbeiter = verdächtig"
- [ ] Genossen verschwinden plötzlich — "ausgewiesen" aus der Stadt
- [ ] Flüstergespräche: Treffen jetzt im Wald, nicht mehr im Lokal

**Mikro — Bürger-Perspektive:**
- [ ] Vater liest am Frühstückstisch die Zeitung: "Endlich Ruhe und Ordnung!"
- [ ] Dienstmädchen weint heimlich — ihr Bruder wurde verhaftet
- [ ] Im Salon: "Diese Sozialisten wollen uns alles wegnehmen"

**Quellen:** 
- Adelheid Popp, "Jugend einer Arbeiterin" (für Arbeiter-Erlebnis)
- Wirsching, "Deutsche Geschichte" S.XX (für Makro)

**Makro:** Staat gegen katholische Kirche. Kanzelparagraph, Schulaufsicht, Jesuitenverbot.

**Mikro — katholische Familie:**
- [ ] Der Pfarrer predigt nicht mehr über Politik — Angst vor Verhaftung
- [ ] Mutter flüstert: "In der Schule lernen sie jetzt protestantische Lieder"

### Gesellschaft

#### Bürgerlicher Alltag — Tagesablauf einer jungen Bürgerin (20 Jahre)
**Quelle:** Marie Witt an Gustav Wegner (Schraut S.23)

| Zeit | Tätigkeit |
|------|-----------|
| ~6:00 | Dienstmädchen weckt, Waschen mit kaltem Wasser |
| ~7:30 | Frühstück mit Familie, Vater liest Zeitung |
| 9:00 | Kaffeetisch abgeräumt |
| 9:00-16:00 | **Montag/Dienstag:** Wäsche ordnen, ausbessern |
| | **Andere Tage:** Handarbeiten, Bruder beaufsichtigen, Klavierüben |
| 16:00 | Mittagessen (Hauptmahlzeit!) |
| Nachmittag | Spaziergang |
| Abend | "Zu meinem Vergnügen" — Klavier, Gesellschaft, Lesen |

**Erlebnis-Details:**
- "Der arme Junge muss dann auch beschäftigt werden" (über den Bruder)
- Implizit: Langeweile, Warten auf Heirat
- Keine bezahlte Arbeit erlaubt!

### Technologie

#### Gasbeleuchtung breitet sich aus
**Makro:** 1878 haben die meisten deutschen Großstädte Gasbeleuchtung. Kleinere Städte folgen.

**Mikro — Erlebnis:**
- [ ] Die Straßen sind nachts nicht mehr stockfinster
- [ ] Der Laternenanzünder kommt jeden Abend — Kinder laufen ihm nach
- [ ] "Früher hat man sich nach Einbruch der Dunkelheit nicht mehr rausgetraut"
- [ ] Im Haus: Gaslampen statt Kerzen/Petroleumlampen = heller, aber Gasgeruch
- [ ] Angst vor Gasexplosionen — Zeitungsberichte über Unglücke
- [ ] Wer kein Gas hat (Arbeiter), sitzt noch bei Kerzenlicht

**Quellen:**
- [ ] Zeitungsarchiv: Berichte über Gasunfälle
- [ ] Technologie-Geschichte
